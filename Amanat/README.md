# Amanat Sekolah
## Aplikasi Peminjaman Peralatan Sekolah berbasis QR-Code berbasis website dengan PHP Native dan MySQL
### Persiapan Sistem
````
Container/Docker :
- php7.4 
- MySQL 5.7
````
### Akses Direktori 
````
+ amanat/
  - index.php
  + uploads/
  + qrcode/
````
````
mkdir -p amanat/{uploads,qrcode}/
chmod -R 0777 amanat/uploads/
chmod -R 0777 amanat/qrcode/
````
### Jalankan
````
Siapkan file index.php  di diretori amanat/
http://ip_address/amanat
Login:
Admin: admin@sekolah.sch.id / 123
````
